# Data folder

This repository does not store the actual data files.

To run this project, place the required input data in this `data/` folder.

Data source:
- [Google Drive](https://drive.google.com/drive/folders/1-JG5AB0wil3ZRgxI0ANRLb5C0X9mJ2nT?usp=sharing)
- The Google Drive is a one-for-one mirror of the 'data/' folder. To run any analysis locally or on the cluster, please add all data located in in the Google Drive into the 'data/' folder, as structured. 

Notes:
- Data files in this folder are ignored by Git.
- Keep filenames consistent with the scripts in this repository.
