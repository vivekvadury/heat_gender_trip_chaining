# Data folder

This repository does not store project data files in Git.

To run the project locally, download the full shared Google Drive folder and copy its contents into this local `data/` directory.

Data source:
- [Google Drive](https://drive.google.com/drive/folders/1-JG5AB0wil3ZRgxI0ANRLb5C0X9mJ2nT?usp=sharing)

Instructions:
- Treat the Google Drive folder as a one-to-one mirror of this `data/` directory.
- Copy all folders and files from Google Drive into `data/`, preserving the folder structure exactly.
- Do not rename files or subfolders unless the project documentation is updated to match.

Notes:
- Data files in this folder are ignored by Git and should not be pushed to GitHub.
- Each collaborator should maintain their own local copy of the shared data in this `data/` directory.
- Run `Rscript scripts/check_data.R` from the repository root after copying data to confirm setup.
